Esse é um sistema simples que está em fase de testes, a ideia base é funcionar apenas como um velocímetro para o jogo F1 22, especificamente desenvolvido para consoles (usando o xbox séries S como base).

Tecnologias:
Esse projeto visa principalmente o uso básico da linguagem Go para interpretar dados UDP via sockets, versões posteriores visam usar java para possíveis updates de interface, seja para transmissão de dados para uma interface em android studio ou algo mais complexo envolvendo uma possível aplicação expo.

Considerações finais:
Mesmo q seja um projeto simples, espero q possa me ensinar mais sobre o protocolo UDP e a própria linguagem Go, escolhida por sua segurança em lidar com altas demandas de dados e sua capacidade relacionada ao baixo nível, para possíveis recrutadores, um aviso, isso é só o começo!
